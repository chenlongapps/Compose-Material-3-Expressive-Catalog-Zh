# Compose Material3 Expressive Catalog

这是一个独立运行的 **Compose Material3 Expressive Catalog**，从 **AndroidX 仓库**中提取而来，用于展示各种 Jetpack Compose 组件和 Material Design 元素。

## 为什么创建这个项目？

Compose Material Catalog 的源代码位于 **AOSP** 中，以便及时跟进最新的 **Compose Material API** 和示例。不过，访问并构建这个应用需要使用体积较大的 **AndroidX** 仓库，过程比较繁琐。

本项目将 AndroidX 源码中的 Catalog 独立提取出来，让开发者无需构建完整的 AndroidX 工程即可探索这些组件和示例。

应用支持 English 和简体中文，可在右上角“更多”菜单中的“语言”选项进行切换。
